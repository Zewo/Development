## Installation

Works best with June 20, 2016 Swift 3 Development Snapshot.

```swift
import PackageDescription

let package = Package(
    dependencies: [
        .Package(url: "https://github.com/QuarkX/Quark.git", majorVersion: 0, minor: 0)
    ]
)
```
